Odoo Studio for Community

The ultimate tool for customizing Odoo without any technical knowledge

Odoo Studio for Community is a powerful application that allows you to easily customize your Odoo instances. With its intuitive interface, you can add new fields to form views, rename existing fields, and rearrange fields using drag and drop. Additionally, you can modify field parameters such as readonly, invisible, required, and many2one, one2many, and more. Keep reading to learn more about these features and discover even more that Odoo Studio has to offer.

Add new fields to form views

One of the key features of Odoo Studio is the ability to easily add new fields to form views. Whether you want to track additional information about your customers, products, or anything else, you can do so with just a few clicks. Simply select the field type you want to add from the list of available options, give it a name, and click "Save". Your new field will appear in the form view, ready for you to use.

Rename existing fields

In addition to adding new fields, you can also rename existing fields in form views using Odoo Studio. This is useful if you want to change the label of a field to something more descriptive, or if you want to use a different language for the field names. To rename a field, simply click on its current name and enter the new name in the field that appears. Your changes will be saved automatically, and the field name will be updated throughout the application

Arrange fields using drag and drop !

Odoo Studio also makes it easy to rearrange the fields in your form views. Simply click and hold on a field, and then drag it to the desired location. The other fields will automatically shift to make room for the moved field, and your changes will be saved automatically. This is a quick and convenient way to customize the layout of your form views to suit your needs.

Modify field parameters.

In addition to adding, renaming, and rearranging fields, you can also modify various parameters of fields using Odoo Studio. For example, you can make a field readonly so that users can view its value but not edit it. You can also make a field invisible if you don't want it to be displayed in the form view. Additionally, you can set a field as required, so that users must enter a value before they can save the form. And with the many2one and one2many options, you can easily create relationships between fields and records. These are just a few examples of the flexibility that Odoo Studio offers when it comes to customizing your form views
